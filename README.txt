Compiling the screen saver:

---------------------Step1---------------------

	Start Visual Basic and load the source code.

-----------------------------------------------
---------------------Step2---------------------

	Click "File" then click "Make *.exe"

-----------------------------------------------
---------------------Step3---------------------

	Goto your prefered folder and enter into the filename box "Mandelbrot Set.scr" and compile

-----------------------------------------------
---------------------Step4---------------------

	Find the compiled file

-----------------------------------------------
---------------------Step5---------------------

	Right Click the file and select Install

-----------------------------------------------
---------------------Step6---------------------

	Configure the screen saver and preview it!

-----------------------------------------------

This Should Now Work As A Screen Saver!
